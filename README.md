# 🚀 InsightAI Analyst

**InsightAI Analyst** is a powerful, next-generation data exploration platform that transforms spreadsheets into intelligent conversations.
Upload your data, ask questions in natural language, and get instant insights with dynamic visualizations.

---

## ✨ Key Features

### 🧠 AI-Powered Data Analysis

* Powered by **Google Gemini 2.0**
* Understands natural language queries
* Automatically cleans and processes datasets

### 📊 Dynamic Visualizations

* Auto-generates interactive:

  * Bar Charts
  * Line Charts
  * Pie Charts
* Built using **Recharts**

### 💎 Premium UI/UX

* Modern **Glassmorphism design**
* Smooth animations & transitions
* Fully responsive interface

### 🔐 Secure Authentication

* Google Sign-In (SSO)
* Integrated with **Firebase Authentication**

### 🌓 Adaptive Theming

* Dark Mode 🌙 / Light Mode ☀️
* Auto-detect system preference
* Persistent theme settings

### 📁 Universal File Support

* Upload:

  * CSV files
  * Excel (.xlsx, .xls)
* Drag & drop support

### ⚡ Quick Demo Mode

* “Sample Data” button for instant testing
* No file upload required

### 💾 Cloud Ready

* Integrated with **Firestore**
* Store:

  * User preferences
  * Analysis history

## 🚀 Getting Started

### 1️⃣ Sign In

Login securely using your Google account.

### 2️⃣ Upload Data

* Drag & drop your CSV/Excel file
  **OR**
* Click on **“Sample Data”**

### 3️⃣ Analyze

Ask questions like:

* *“Show sales by region”*
* *“Give me top 5 products”*
* *“Summarize key trends”*

### 4️⃣ Visualize

* Get instant charts and insights
* Interactive and easy to understand

### 5️⃣ Export

* Download processed or analyzed data as CSV
